javascript闭包

12月 30, 2014 分類 javascript

闭包的概念很难懂，这篇文章也只是我的一个学习笔记吧，其中尤其是闭包在循环中的运用尤其最难理解。
综合来说的:闭包是一个函数，在函数的上下文中有变量由于被引用，即使函数返回也不会销毁，这样的函数就是一个闭包。
